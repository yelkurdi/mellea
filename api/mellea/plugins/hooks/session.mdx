### <span className="ml-2 inline-flex items-center rounded-full px-2 py-1 text-[0.7rem] font-bold tracking-wide bg-[#4ADE8033]/20 text-[#15803D]">CLASS</span> `SessionPreInitPayload` <sup><a href="https://github.com/generative-computing/mellea/blob/v0.7.0.dev0/mellea/plugins/hooks/session.py#L13" target="_blank"><Icon icon="github" style="width: 14px; height: 14px;" /></a></sup>


Payload for `session_pre_init` â€” before backend initialization.

**Attributes:**
- `backend_name`: Name of the backend (e.g. `"ollama"`, `"openai"`).
- `model_id`: Model identifier string (writable).
- `model_options`: Optional dict of model options like temperature, max_tokens (writable).
- `context_type`: Class name of the context being used (e.g. `"SimpleContext"`).

### <span className="ml-2 inline-flex items-center rounded-full px-2 py-1 text-[0.7rem] font-bold tracking-wide bg-[#4ADE8033]/20 text-[#15803D]">CLASS</span> `SessionPostInitPayload` <sup><a href="https://github.com/generative-computing/mellea/blob/v0.7.0.dev0/mellea/plugins/hooks/session.py#L29" target="_blank"><Icon icon="github" style="width: 14px; height: 14px;" /></a></sup>


Payload for `session_post_init` â€” after session is fully initialized.

**Attributes:**
- `session_id`: UUID string identifying this session.
- `model_id`: Model identifier used by the backend (e.g. `"granite4.1\:3b"`).
- `context`: The initial [`Context`](../../core/base#class-context) instance for this session.
